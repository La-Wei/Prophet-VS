# Autocostruzione ricambi

Questa cartella raccoglie materiale per un tecnico che voglia valutare la ricostruzione di ricambi non reperibili.

Aprire prima:

1. [CEM5530 / ELD5530: dossier per clone o ricostruzione](cem5530_clone_build_dossier.md)
2. [Progetto preliminare ELD5530 V2-like con MAX5167](max5167_v2_like_pre_design.md)
3. [Rev A MAX5167 - prototipo non validato](rev_a_max5167_prototipo/README.md)
4. [Contatti per acquisto ELD5530 / CEM5530 replacement](contatti_acquisto_eld5530_cem5530.md)

Nota: questa cartella non contiene Gerber o schema ELD5530 originale, perche' non ne ho trovato una pubblicazione verificabile. Il dossier serve a evitare lavoro alla cieca; il progetto `MAX5167` porta invece la pista piu realistica fino a pin mapping, alimentazioni, vincoli PCB e test minimi. Synthelectro 2014 conferma IC Maxim Integrated per `ELD5530 V2`, ma non pubblica la sigla: `MAX5167` resta quindi un'ipotesi tecnica forte, non una BOM ufficiale. La sottocartella `rev_a_max5167_prototipo/` e' una base concreta per CAD/prototipo, ma resta non validata e non va montata nel Prophet VS senza collaudo fuori macchina.
